# Short-Form Test Posting Matrix

**Status:** ready for execution
**Date:** 2026-05-17
**Primary goal:** compare opening-hook strength under roughly comparable posting conditions
**Primary candidate order:** `Name on Neon` → `Leave the Light` → `Back in Color`
**Metric sink:** `docs/shortform/sprint1-hook-test-metrics.csv`

## Test principle

Do not vary everything at once. For this first pass, keep:
- similar clip length range
- similar caption length
- similar posting windows where possible
- one dominant visual idea per candidate
- one clear emotional framing per post

The objective is not broad campaign rollout yet. The objective is to learn which hook survives the first-scroll test best.

## Candidate matrix

| Candidate | Clip slug | Primary angle | First-frame visual | Caption posture | What success looks like | Main kill signal |
|---|---|---|---|---|---|---|
| Name on Neon | `promo-name-on-neon-direct-chorus` | glow-up / visible self-worth | neon sign, mirror, city-night reveal | declarative / quotable | immediate lyric echo and strong replay instinct | feels slogan-only with weak emotional payoff |

## Suggested posting windows

Use local audience-friendly windows and try to keep them comparable:
- Window A: 12:00-2:00 PM local
- Window B: 6:00-9:00 PM local
- Window C: 9:00-11:00 PM local

Recommended first pass:
1. `Name on Neon` in Window B
2. `Leave the Light` in Window C
3. `Back in Color` in Window B or A

If possible, avoid posting all three back-to-back within the same hour.

## Platform-by-platform framing

### TikTok / Shorts / Reels baseline

Use the same core asset logic across platforms:
- opening hook visible in frame 1
- on-screen lyric text for the first key line
- no long setup before the hook
- keep visual motion active but not noisy

### Candidate 1 — Name on Neon

- **Visual direction:** neon sign, makeup mirror, streetlight walk, backstage glow-up
- **Text-on-screen options:**
  - `got my name on neon`
  - `burning where the doubt was`
  - `the breakup glow-up song`
- **Best use case:** lead public test because the image/title lock is immediate
- **Warning:** do not oversell empowerment so hard that the emotional wound disappears

### Candidate 2 — Leave the Light

- **Visual direction:** porch light, text thread, parked car, blurry windshield, waiting outside
- **Text-on-screen options:**
  - `say don't come over`
  - `leave the light`
  - `i'm already outside`
- **Best use case:** late-night emotional lane test
- **Warning:** the silence-drop must feel intentional, not like weak pacing

### Candidate 3 — Back in Color

- **Visual direction:** grayscale-to-color transition, stage relight, outfit change, nightlife return
- **Text-on-screen options:**
  - `now i'm back in color`
  - `back where the bright lights are`
  - `the comeback after the gray phase`
- **Best use case:** mainstream broad-audience fallback test
- **Warning:** needs strong visual execution or it risks feeling too generic

## Logging template per post

For every upload, log these alongside the CSV:
- platform
- account
- post time
- candidate slug
- visual framing used
- text-on-screen used
- caption version used
- first 24h completion signal
- first 24h replay/rewatch signal
- comments showing quote-back recall
- creator-reuse or duet/remix signal if any
- advance / rerun / kill decision

## Decision priority

When metrics split, prioritize in this order:
1. quote-back recall
2. rewatch / replay intent
3. save / revisit intent
4. completion
5. broad like-volume

A clip that gets quoted and replayed beats a clip that gets shallow passive reach.
