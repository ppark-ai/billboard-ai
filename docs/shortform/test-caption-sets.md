# Short-Form Test Caption Sets

**Status:** ready for posting
**Date:** 2026-05-17
**Use with:** `docs/shortform/test-posting-matrix.md`

## Usage rule

For the first test pass, keep captions short and single-angle. Do not stack multiple ideas in one caption. The caption should reinforce the clip's emotional thesis, not explain the whole song.

---

## Candidate 1 — Name on Neon

### Angle A — glow-up / self-worth
1. got my name on neon and suddenly the doubt looks small
2. breakup aftermath but make it bright enough to see from across the street
3. when the line hits like you finally belong in your own life again

### Angle B — quoteable lyric push
1. got my name on neon / burning where the doubt was
2. this line feels like the exact second the switch flips back on
3. if the hook is the caption, that's usually a good sign

### Angle C — creator-friendly prompt
1. the glow-up reveal sound
2. use this for the before/after that changed everything
3. the city-night comeback clip song

### Best first caption
- `got my name on neon / burning where the doubt was`

---

## Cross-candidate control captions

Use these only if you want a more apples-to-apples comparison across candidates:
- `which opening hits first?`
- `testing the hook before the full release`
- `which one would you replay?`

These are useful for controlled comparison but weaker for natural audience behavior, so prefer native emotional captions first.

## Recommendation

First posting pass:
1. `Name on Neon` → `got my name on neon / burning where the doubt was`
2. `Leave the Light` → `say don't come over / i'm already outside`
3. `Back in Color` → `now i'm back in color`

Those choices keep each candidate aligned to its clearest emotional and visual lock.
