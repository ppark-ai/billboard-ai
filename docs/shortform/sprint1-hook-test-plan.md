## Test objective

Determine which promoted hook has the strongest short-form opening and the clearest path to public test packaging.

## Primary metrics to collect

- completion rate
- rewatch intent
- save / revisit intent
- quote-back recall in comments or reviewer notes
- visual pairing strength
- creator reuse potential

## Evaluation method

### Phase 1 — internal pass
- verify each promoted clip is playable and maps cleanly to the intended title/hook framing
- confirm no promoted clip feels weaker than its sprint-1 parent concept

### Phase 2 — lightweight audience test
- post or privately share 1-3 cuts per candidate
- compare which opening survives the first-scroll test best
- collect qualitative notes on lyric recall and comment language

### Win rule

A candidate advances if:
1. its clip opening is clearly memorable within the first 3-5 seconds
2. it beats the other promoted cuts on replay / save-intent signals
3. its title, lyric, and visual angle all point in the same direction
