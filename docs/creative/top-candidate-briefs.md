## Candidate 1 — Name on Neon

- **Source demo:** Lane 1 Demo 1C
- **Slug:** `lane1-demo1c-name-on-neon`
- **Title lock:** **Name on Neon**
- **Audience lane:** recovery-pop / glow-up anthem / female-front radio-pop
- **Emotional thesis:** self-worth becomes public and visible after heartbreak
- **Core hook line:** “Got my name on neon / Burning where the doubt was”
- **Why promoted:** fastest opening, strongest visual metaphor, easiest clip hook under 10 seconds

### Short-form angles
1. **Direct chorus open** — start with “Got my name on neon” immediately
2. **Glow-up reveal** — start on the line “Burning where the doubt was” with visual transformation
3. **Post-chorus loop** — “Neon, neon” loop for repeatability and creator reuse

### Visual framing
- neon signage
- city-night confidence walk
- before/after breakup transformation
- mirror / makeup / backstage / streetlight palette

### Production priorities for next pass
- brighter stereo width than sprint-1 baseline
- immediate vocal clarity in the first 2 seconds
- stronger kick lift entering the first hook
- keep phrasing clean and slogan-like, avoid over-singing

### Kill criteria
- if the hook feels too slogan-first and not emotional enough
- if “neon” repeats without enough payoff in the next line

## Candidate 2 — Leave the Light

- **Source demo:** Lane 2 Demo 2A
- **Slug:** `lane2-demo2a-leave-the-light`
- **Title lock:** **Leave the Light**
- **Audience lane:** intimate nocturnal pop / confessional alt-pop / late-night TikTok emotional lane
- **Emotional thesis:** saying “don’t” while already crossing the line back into the relationship
- **Core hook line:** “Leave the light, leave the light” + “I’m already outside”
- **Why promoted:** best tension/release mechanic and strongest built-in silence-drop moment

### Short-form angles
1. **Lyric-first open** — cold-open on “Say don’t come over / Leave the light”
2. **Silence-drop open** — cut to silence, then hit “I’m already outside”
3. **Close-mic confession** — minimal intro, almost spoken tension before the hook lands

### Visual framing
- midnight driveway / porch light
- text-message regret / impulsive pull-up
- blurry windshield / parking outside the house
- dark hallway / lit doorway contrast

### Production priorities for next pass
- preserve intimacy; don’t overfill the arrangement
- exaggerate the silence-drop transition
- keep the vocal as close-mic and vulnerable as possible
- allow low-end pulse without crowding the hook phrase

### Kill criteria
- if the tension collapses before the drop
- if the title line sounds passive instead of obsessive / impulsive

## Candidate 3 — Back in Color

- **Source demo:** Lane 1 Demo 1A
- **Slug:** `lane1-demo1a-back-in-color`
- **Title lock:** **Back in Color**
- **Audience lane:** mainstream comeback pop / empowering recovery anthem
- **Emotional thesis:** the breakup removed the grey filter; the self returns in full saturation
- **Core hook line:** “Now I’m back in color / Back where the bright lights are”
- **Why promoted:** broadest commercial framing and easiest mainstream lane positioning

### Short-form angles
1. **Direct payoff open** — “Now I’m back in color” right at frame one
2. **Shadow-to-color transition** — use “You lost me in the shadows / I found me in the dark” as setup into payoff
3. **Big final chorus clip** — lean into larger drums and stacked vocals

### Visual framing
- desaturated-to-saturated transition
- club / street / stage relight
- breakup comeback montage
- wardrobe / confidence / nightlife restoration

### Production priorities for next pass
- widen chorus hit without washing out lyric clarity
- maintain a clean, major-key lift
- emphasize drum arrival and vocal stack on payoff
- avoid generic EDM overbuild; stay pop-first

### Kill criteria
- if the phrase reads too generic compared to `Name on Neon`
- if the production gets big before the hook earns it
