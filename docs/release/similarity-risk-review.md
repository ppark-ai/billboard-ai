## Review method

This is an internal pattern-level review only. It checks for:
1. title uniqueness risk inside the current repo outputs
2. overt mimicry / artist-specific framing in the briefs
3. overly generic topline phrases that may need strengthening
4. lyrical red flags that imply quotation, parody, or direct reuse
5. lane-overfit risk where a candidate feels too template-like to justify release work

This review is **not** a legal clearance opinion and does **not** replace external catalog search or counsel if the project moves toward release.
